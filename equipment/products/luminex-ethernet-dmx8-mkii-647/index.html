Luminex Ethernet-DMX8 MkII

The Ethernet-DMX8 MkII is an Ethernet node that serves eight DMX512 outlets, compatible with the Artnet and sACN protocols. All outlets are conform to DMX512-A and can be used as input or output.
The two Ethernet links on the front and at the rear are 10/100BaseTX connections on Neutrik RJ45 Ethercon connectors. The Ethernet-DMX8 MkII embeds a built-in switch fabric which allows to use the converter as an Ethernet repeater.
All configuration can be done over Ethernet through a build in Web site. This all comes in a 19" 1 unit high metal housing that can be rack mounted.
