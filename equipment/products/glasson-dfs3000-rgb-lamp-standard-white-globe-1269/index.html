Glasson DFS3000 RGB Lamp Standard - White Globe

Manufactured from robust UV resistant Acylic with a standard ES (E27) lamp base the LED unit comprises RGB or RGB + W LEDs with an advanced microcontroller based linear current control system to provide exceptional LED control



The combined power and data system allows up to 170 RGB or 120 RGBW units to be individually controlled on a common two conductor cable

​

Note that the cable does not need to be a single ‘festoon’ type connection. LEDs can be wired individually or in groups back to the power unit with common or individual feeds.



Basically providing the LEDs are wired in parallel and a given cable feed length does not exceed 100M the system will work!

The cable type is not critical however 6A 60V rating is required. There are three intensity versions. Std, Hi and SH
